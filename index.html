三層 Agent 架構

Agent A · 初篩

每週自動從 arXiv 抓取 cs.LG 與 cs.CR 最新論文，以 Llama-3 極速過濾，只留下具備實質突破的篇目。

Agent B · 深析

以 Gemini 2.5 Flash 對精選論文進行深度解析，萃取核心技術貢獻與兩個可行的後續研究方向。

Agent C · 跨域

在資安、統計、通訊、機器學習之間尋找底層數學同構性，建構可互動的跨域知識圖譜。
